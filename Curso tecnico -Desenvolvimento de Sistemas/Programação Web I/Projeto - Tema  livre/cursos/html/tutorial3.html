Estrutura Básica de um Documento HTML

Anatomia de uma Página Web

Todo documento HTML segue uma estrutura padrão para que os navegadores possam interpretá-lo corretamente. Conhecer essa estrutura é o primeiro passo para criar qualquer página web. Ela é composta por um conjunto de tags essenciais que definem o tipo de documento, o idioma e a divisão entre o cabeçalho e o corpo.

A primeira linha de um documento HTML é o <!DOCTYPE html>. Essa declaração informa ao navegador que o documento é uma página HTML5. Logo em seguida, a tag <html> define o início e o fim de todo o documento e, geralmente, inclui o atributo "lang" para especificar o idioma, como "lang="pt-br"".

Dentro da tag <HTML>, existem duas seções principais: o <head> e o <body>. A tag <head> contém metadados e informações sobre a página que não são exibidas diretamente, como o título que aparece na aba do navegador (<title>), a codificação de caracteres (<meta charset="UTF-8">) e o link para arquivos externos de estilo (<link rel="stylesheet">).

O <body> é a parte visível da página. Ele contém todo o conteúdo que o usuário vê e interage, como textos, imagens, botões, vídeos e links. É dentro da tag <body> que você escreve a maior parte do seu código HTML, estruturando o conteúdo da sua página.
