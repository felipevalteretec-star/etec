Estrutura do Corpo (Body) de um Documento HTML

Elementos Visíveis da Página

O <body> é a parte mais importante do documento HTML, pois é nele que o conteúdo que será exibido para o usuário é colocado. Todo o texto, imagens, vídeos, tabelas, e outros elementos interativos que você vê em uma página web estão aninhados dentro da tag "<body>".

Para organizar o conteúdo de forma semântica e acessível, o HTML5 introduziu novas tags que dividem o "<body>" em seções lógicas. A tag <header> é usada para o cabeçalho principal da página, que geralmente inclui o título do site e a navegação. A tag <nav>, que fica dentro do "<header>", é especificamente para links de navegação.

A tag <main> delimita o conteúdo principal e único da página. Dentro do "<main>", você pode usar tags como <section> para agrupar conteúdos relacionados e <article> para artigos e posts independentes. Para barras laterais de conteúdo secundário, a tag <aside> é a mais apropriada.

Por fim, a tag <footer> é utilizada para o rodapé da página. Geralmente, ela contém informações como direitos autorais, links para contato e mídias sociais. Usar essas tags corretamente não apenas organiza o seu código, mas também melhora a acessibilidade e a otimização para motores de busca (SEO).
