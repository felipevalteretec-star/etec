Historia do Html

O HTML (HyperText Markup Language) surgiu no início da década de 1990, criado por Tim Berners-Lee, um físico do CERN (Organização Europeia para a Pesquisa Nuclear). Na época, o grande desafio era a falta de um sistema simples e universal para compartilhar documentos científicos entre pesquisadores de diferentes instituições, que usavam computadores e softwares incompatíveis.

A motivação por trás do HTML foi justamente resolver essa fragmentação. Berners-Lee precisava de uma linguagem que pudesse ser entendida por qualquer computador, em qualquer lugar do mundo, para que a informação pudesse ser facilmente ligada a outras informações. Ele imaginou um sistema de "hipertexto", onde documentos pudessem ser interconectados por meio de links.

O HTML resolveu o problema de compatibilidade e acesso universal. Ele permitiu que a web se tornasse um ecossistema de documentos interligados, onde um simples clique em um link leva o usuário de uma página a outra. Sua sintaxe baseada em tags (<tags>) deu uma estrutura básica e comum para a criação de documentos, tornando a distribuição e a visualização de conteúdo acessível para todos, independentemente do sistema operacional ou navegador.
