Estrutura de um Script JavaScript

Onde colocar o JavaScript?

O JavaScript pode ser escrito diretamente dentro de uma página HTML usando a tag <script>. Geralmente, ele é colocado antes da tag </body> para garantir que todo o HTML seja carregado antes da execução do script. Também é possível criar arquivos externos com a extensão .js e importá-los usando o atributo src.

Exemplo básico com DOM

Neste exemplo, temos um botão que, ao ser clicado, altera o texto de um parágrafo. 

            <!DOCTYPE html>
            <html>
            <head>
                <title>Exemplo JavaScript</title>
            </head>
            <body>
                <p id="mensagem">Texto original</p>
                <button onclick="mudarTexto()">Clique aqui</button>

                <script>
                    function mudarTexto() {
                        document.getElementById("mensagem").innerText = "Texto alterado com JavaScript!";
                    }
                </script>
            </body>
            </html>

Aqui usamos a função document.getElementById() para acessar um elemento HTML e alterar seu conteúdo usando innerText. Essa é a base da manipulação do DOM com JavaScript.
