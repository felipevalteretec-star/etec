Variáveis e Tipos de Dados em JavaScript

O que são variáveis?

Em JavaScript, variáveis são espaços na memória que guardam valores. Elas podem armazenar números, textos, booleanos, objetos, arrays, entre outros tipos de dados. Para criar variáveis, usamos as palavras-chave var, let ou const.

Tipos de Dados

Os principais tipos de dados em JavaScript são: Number (números), String (texto), Boolean (verdadeiro/falso), Object (objetos), Array (listas), Null e Undefined.

Exemplo prático

            // Declarando variáveis
            let nome = "Valter";       // String
            let idade = 30;            // Number
            let estudante = true;      // Boolean

            // Exibindo no console
            console.log("Nome:", nome);
            console.log("Idade:", idade);
            console.log("É estudante?", estudante);

Neste exemplo, criamos três variáveis: uma string, um número e um booleano. O método console.log() exibe as informações no console do navegador.
